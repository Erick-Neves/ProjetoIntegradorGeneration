Milhares de artistas freelancers prontos para começar a trabalhar

Como funciona:

Demanda:

Conte-nos como você gostaria que o artista atuasse para você.

Selecione:

Receba propostas e defina qual artista você irá contratar.

Contratar:

Agende o horário desejado, e pague o valor hora correspondente.

Finalize:

Após o trabalho finalizado, a plataforma libera o pagamento total para o artista.

Contrate serviços como:

Mágicos para vislumbrar sua audiência.

Grafiteiros para trazer vida às suas superfícies.

Músicos para animar suas festas.

Dançarinos de vários estilos para aquela festa de debutante.

Atores de teatro para colocar suas ideias em prática.

Palhaços para divertir os seus eventos.
